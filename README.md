# Terminal TicTacToe

Simple TicTacToe in the terminal for two human players. First try at programming a Java application.

## How to play

After compiling, just run

```bash
  java Game
```

  
## Running Tests

To run tests, run the following command

```bash
  java Test
```


## Compiling

To compile, just run

```bash
  javac *.java
```
